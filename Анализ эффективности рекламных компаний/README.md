## Анализ эффективности рекламных компаний в социальных сетях
Источник датасета - https://www.kaggle.com/datasets/alperenmyung/social-media-advertisement-performance

# Используемые инструменты:
PostgreSQL, Power BI

# Цель проекта:
Проанализировать успешность проведенных рекламных компаний за период.
Сформировать рекомендации для улучшения таргетинга и перенаправления бюджета в наиболее эффективные каналы продвижения.

# Задачи:
- рассчитать общие основные метрики (кол-во пользователей, показов, кликов, лайков, общая конверсия по рекламе, общие затраты на рекламу)
- рассчитать основные метрики и метрики эффективности (CTR, Conversion Rate, CPC, CPM) отдельно для каждой рекламной компании
- проанализировать аудиторию разрезе различных признаков (пол, география, возраст)
- провести анализ лидогенерации и попадания рекламы в целевую аудиторию
- провести анализ эффективности каналов продвижения 
- покупательское поведение в разрезе географии и пола

# Рассчитанные метрики:
На каждом дашборде метрики рассчитаны по единой методике.
Целевым действием считается покупка.

Conversion Rate = Покупки/Количество пользователей
CTR = Клики/Показы
CPM = бюджет рекламной компании/ кол-во просмотров *1000
CPC = бюджет рекламной компании/кол-во кликов
CAC = бюджет рекламной компании/кол-во привлеченных клиентов
ER (Engagement Rate) = (лайки + комментарии + репосты)/охват

# Описание данных:
Таблица Ad Events

| Столбец     | Описание    | 
|-------------|-------------|
| Event_id    | Уникальный идентификатор для каждого взаимодействия  |
| Ad_id       | Внешний ключ, ссылающийся на таблицу объявлений  |
| User_id     | Внешний ключ, ссылающийся на таблицу пользователей  |
| Timestamp   | Дата и время взаимодействия  |
| Day_of_week | Название дня недели, когда произошло событие  |
| Time_of_day | Период времени суток, когда произошло событие  |
| Event_type  | Тип взаимодействия |

Таблица Ads

| Столбец     | Описание    | 
|-------------|-------------|
| Ad_id       | Уникальный идентификатор для каждого объявления  |
| Campaign_id | Внешний ключ, ссылающийся на родительскую таблицу кампаний  |
| Ad_platform | Платформа, на которой размещается реклама  |
| Ad_type     | Формат рекламы (Изображение, видео …)  |
| Target_gender | Целевой пол аудитории  |
| Target_age_group | Целевая возрастная группа аудитории  |
| Event_type  | Тип взаимодействия |

Таблица Campaigns

| Столбец     | Описание    | 
|-------------|-------------|
| Campaign_id | Уникальный идентификатор для каждой кампании  |
| Name        | Название компании  |
| Start_date  | Дата начала компании  |
| End_date    | Дата окончания компании  |
| Duration_days | Продолжительность компании в днях  |
| Total_buget | Общий бюджет компании  |

Таблица Users

| Столбец     | Описание    | 
|-------------|-------------|
| User_id	    | Уникальный идентификатор для каждого пользователя |
| User_gender | Пол пользователя, указанный в аккаунте |
| User_age    | Возраст пользователя, указанный в аккаунте  |
| Age_group	  | Принадлежность пользователя к возрастной группе  |
| Country     | Страна пользователя  |
| Location    | Точное местоположение или город пользователя  |
| Interests   | Список интересов пользователей, разделенных запятыми, для таргетинга  |
